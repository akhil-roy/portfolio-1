ECommerce platform - personal project

Developed a robust and feature-rich E-commerce platform as a personal project, designed to also function as a boilerplate for client applications. The platform is built using the Django framework for backend development, ensuring scalability and reliability, while the frontend utilizes HTML, CSS, and JavaScript for a clean and responsive design. Key features include User and Admin authentication and authorization, enabling secure access and role-based controls. Integrated categorized product listings for efficient navigation and enhanced user experience. Additional functionalities such as a shopping cart, wishlist, and secure payment processing contribute to a seamless shopping experience. The platform is fully responsive, ensuring accessibility and usability across all devices. This project not only highlights expertise in web development but also demonstrates a strong understanding of user-centric design and functionality. It serves as a testament to delivering flexible and customizable solutions for diverse client needs.
